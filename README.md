# Iris Flower Classification using K-Nearest Neighbors

## Overview

This project implements a supervised machine learning model to classify Iris flowers into three species: Setosa, Versicolor, and Virginica using the K-Nearest Neighbors (KNN) algorithm.

The model is trained on the famous Iris dataset and evaluated using various performance metrics.

## Features

- Data preprocessing and feature scaling
- Train-test data splitting
- K-Nearest Neighbors classification
- Accuracy calculation
- Confusion Matrix generation
- Precision, Recall, and F1-Score evaluation
- Classification Report
- Optimal K-value selection using error rate graph
- Prediction of species for new flower samples

## Dataset

The project uses the Iris dataset containing 150 samples with four features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target classes:

- Setosa
- Versicolor
- Virginica

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn

## Performance Metrics

The model is evaluated using:

- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1 Score
- Classification Report

## Workflow

1. Load Iris dataset
2. Apply feature scaling
3. Split data into training and testing sets
4. Train KNN model
5. Make predictions
6. Evaluate model performance
7. Visualize error rates for different K values
